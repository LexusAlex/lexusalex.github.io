Как сделать pull request проекта

1. Сделать форк проекта на странице его на гитхаб https://github.com/doka-guide/content
2. Клонировать форк проекта на компьютер clone git clone git@github.com:LexusAlex/content.git
3. Запустить проект, узнать как это сделать, если ли тесты
4. Либо можно подключить оригинальный репозиторий к нашему проекту git remote add upstream
5. Или скачивать обновления в репозиторий прямо на своей странице форка
6. пишем код в отельной ветке, правим, и отправляем его в наш форк
7. Создаем pull request и отправляем его авторам
