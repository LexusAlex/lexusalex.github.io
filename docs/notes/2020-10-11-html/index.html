HTML. Контекст использования

Контекст использования тегов html

Категории контента

Элементы html разбиты на категории и могут относиться к одной, к нескольким категориям или могут быть без категории.

Потоковый контент

Основная группа элементов это Flow content. В нее входят большинство элементов

Потоковое содержимое - это элементы, которые находятся в элементе <body>. Например элемент <code>

Спецификация

Явный контент

Обособленная дополнительная группа элементов называется Palpable content.

Означает, что элемент должен быть заполнен контентом. Это не обязательное условие, так как этот элемент может быть подготовлен пустым для последующего заполнения дочерними элементами или заполнением через скрипты, но рекомендуется не оставлять его пустым. Судя по спецификации таких элементов очень много. Например элемент <div>

Фразовый контент

Элементы для оформления текста входят в группу Phrasing content. Она содержит элементы размечающие текст элементы на уровне абзаца.

Большинство элементов типа Phrasing content могут содержать только элементы этого же типа.

Текстовый узел является так же фразовым контентом. Пример элемент <span>

Спецификация

Элементы

  • Заголовки h1, h2, h3, h4, h5, h6

Заголовки h1, h2, h3, h4, h5, h6

Заголовки используются в первую очередь для создания структуры текста и определяют смысловой заголовок раздела.

Категории

  • Flow content
  • Heading content
  • Palpable content

Дочерние категории

  • Phrasing content

Элемент header не является заголовочным, хотя скорее всего содержит контент такого типа.

Особенности

  • Заголовки различают по рангам от h1 до h6.
  • Желательно чтобы заголовок h1 был один на странице, это важно для поисковых систем и экранных читалок.
  • Сейчас рекомендуют вернуться к сквозной иерархии заголовков на странице.
  • Не пропускайте уровни заголовков на странице.
  • Внутри тега article или section заголовки можно начинать с h1, но не рекомендуется.
  • Не стоит определять ранг заголовка по размеру текста на странице, крупный текст это не всегда заголовок.

Стили по умолчанию

У каждого элемента html в браузере определены стили по умолчанию. Их рассмотрение выходит за рамки данной статьи.

h1 {
    display: block;
    font-size: 2em;
    margin: 0 0.67em 0 0.67em;
    font-weight: bold;
   }
h2 {
    display: block;
    font-size: 1.5em;
    margin: 0 0.83em 0 0.83em;
    font-weight: bold;
    }
h3 {
    display: block;
    font-size: 1.17em;
    margin: 0 1em 0 1em;
    font-weight: bold;
}
h4 {
    display: block;
    margin: 0 1.33em 0 1.33em;
    font-weight: bold;
}
h5 {
    display: block;
    font-size: .83em;
    margin: 0 1.67em 0 1.67em;
    font-weight: bold;
}
h6 {
    display: block;
    font-size: .67em;
    margin: 0 2.33em 0 2.33em;
    font-weight: bold;
}

Структурный элемент article

Категории

  • Flow content.
  • Sectioning content.
  • Palpable content.

Дочерние категории

  • Flow content.

Особенности

  • Независимая, отделяемая смысловая часть страницы или приложения.
  • Внутри важен заголовок
